CSCB20: Introduction to Databases and Web Applications

CSCB20H3: Introduction
to Databases and Web
Applications

This course introduces databases and web app development, covering
database terminology, querying, updates, and entity-relationship design.
It also explores static and interactive web documents, dynamic server
content, and integrating databases with web applications.
