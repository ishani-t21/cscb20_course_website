Assignments

Here you can find all the assignments for the course. Make sure to submit them before the due date! Stay updated with the latest tasks.

Assignment 1: Relational Algebra and MySQL

Group Assignment

Due Date: March 20, 2025

Weight: 10%

Overview: Work in a team to design and execute advanced SQL queries to extract, analyze, and modify data in a relational database. The assignment includes writing complex queries using JOINs, subqueries, and aggregate functions.

Technologies:

  • MySQL
  • SQLite

Skills Learned:

  • SQL syntax
  • Database queries
  • Schema design
  • Optimization

Assignment 2: Web Application and Database

Individual Assignment

Due Date: April 02, 2025

Weight: 15%

Overview: Build a dynamic web application that allows users to explore details about a novel of your choice. The app should include an interactive book summary, character profiles, and user reviews. You will implement a database to store book-related data, integrate JavaScript for interactivity, and style the interface using CSS frameworks.

Technologies:

  • HTML
  • CSS
  • JavaScript
  • React.js
  • Node.js
  • SQL
  • REST API

Skills Learned:

  • Front-end development using HTML, CSS, and JavaScript
  • Creating responsive and visually appealing UI
  • Basic database integration
  • Handling user interactions and dynamic content
