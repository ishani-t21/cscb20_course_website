Labs / Tutorials

Here you can find all the lab exercises for the course. Make sure to submit them before the due date! Stay updated with the latest tasks. Note that tutorials start from week 2.

Lab 1: RA Queries

Group Lab

Due Date: March 20, 2025

Weight: 10%

Overview: Learn the basics of relational algebra, a formal query language used to manipulate and retrieve data from databases. Practice using operations like selection (σ), projection (π), union (∪), intersection (∩), and join (⨝) to process data efficiently.

Technologies:

  • Relational algebra notation
  • Database query tools

Skills Learned:

  • Understanding relational algebra operations
  • Translating SQL queries into relational algebra
  • Manipulating database relations

Lab 2: Web App Development

Individual Lab

Due Date: April 02, 2025

Weight: 15%

Overview: Design a fully responsive web page using CSS Grid and Flexbox. Ensure the layout adapts smoothly to different screen sizes and devices. Implement media queries for optimized mobile and tablet views.

Technologies:

  • HTML
  • CSS (Flexbox & Grid)
  • Media Queries

Skills Learned:

  • Mobile-first design
  • CSS responsiveness
  • Adaptive layouts
